Установить Python v3.10
Установить autohotkey

Установить библиотеки (ПКМ на меню Пуск -> Выполнить -> cmd:
pip install mss
pip install opencv-python
pip install ahk

В файлах bot и main в параметре execute_path прописать путь до файла AutoHotKey.exe для этого пишем в поиске винды AutoHotKey.exe и нажимаем путь до файла
Открываем браузер, заходим в игру, в руке должна быть удочка.
Запускаем файл start.py!
После запуска файла открываются консоли нажимаем на игру чтобы окно браузера было активно и оставляем бота он будет рыбачить до тех пор пока не сломается удочка
